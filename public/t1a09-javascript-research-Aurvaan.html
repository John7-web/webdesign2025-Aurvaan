What is javascript


it is a programming language

Fundamental of javascript

  • classes - provide a structured way to create objects with shared properties and methods
  • data types - nature of data stored in variables, affecting how values are processed and interacted with in the code
  • Variables - it is the basic unit of storage in a program
  • Functions - reusable blocks of code designed to perform specific tasks
  • Loops - repeats a block of code a specific number of times
  • Arrays - an array is an ordered list of values
  • Objects - dynamic data structure that stores related data as key-value pairs, where each key uniquely identifies its value
  • Operators - symbols or keywords used to perform operations on values and variables
  • Type Conversion - defined as converting the data type of the variables from one type to the other manually by the programmer(explicitly) or automatically by the JavaScript(implicitly)
  • Events - actions or occurrences that happen in the browser
  • double equal - operator tests for abstract equality
  • The triple equal - operator tests for strict equality
